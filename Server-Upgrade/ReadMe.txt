Information To Keep Note Of:
Assumes you are running the jar as "Thermos-server.jar"
Deletes any file called "Thermos.1.7.10<everything in-between>.jar"

Requirements:
Unix System or Windows with a program equal or equivalent to Cygwin
Both of the following in the same directory:
options-run.sh
do-not-edit.sh

Running:
Edit options-run.sh
Then ./options-run.sh
